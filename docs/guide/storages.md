# 添加存储

你需要将音声作品以文件夹的形式分类好，每个作品对应一个文件夹，且**文件夹名称必须包含作品 ID**，示例：

```
D:\VOICES
├─RJ395908
└─[みやぢ屋][RJ334212]ガチ恋不可避の耳リフレ2~ぼくっこ店員ゆずるの出張サービス~
```

![add-storage](/images/add-storage.webp)

存储添加成功后请点击【扫描】按钮，Neokikoeru 会在后台自动完成作品刮削并录入到媒体库，你可以前往【任务】页面查看进度。

> [!NOTE]
> 重复的作品会覆盖，且顺序是不确定的。

## 通用项

### 最大扫描深度

如果识别到作品文件夹，程序会提前结束当前目录的扫描，否则继续向下扫描其子目录，直到遍历层级超过【最大扫描深度】设置的值。

### 忽略文件夹

如果你希望扫描时跳过某些目录，你可以设置【忽略文件夹】，支持通配符。例如 `_*` 表示忽略以下划线 `_` 开头的文件夹，包括其子目录。

### 强制代理

当存储驱动为网盘时，如果上游支持，Neokikoeru 默认会通过重定向的方式访问文件资源，不消耗服务器流量。

但某些网盘可能不支持跨域访问，或者返回的 MIME 类型不正确，导致浏览器无法阅览/播放，这种情况下你可以启用【强制代理】，使用服务器进行中转。

## 本地存储

### 根文件夹路径

目标文件夹的绝对路径，如 `/root` 。

## 百度网盘

### 根文件夹路径

以 `/` 开头的绝对路径，根目录为 `/` 。

## OneDrive

### 根文件夹路径

以 `/` 开头的绝对路径，根目录为 `/` 。

## Google Drive

### 根文件夹 ID

文件夹的唯一标识，可从浏览器地址栏获取。

目标文件夹对应 URL `https://drive.google.com/drive/folders/1Qd9RC5Lr0J7VHUssBSv5sKDhxnla_GSs` 最后面的一串字符
`1Qd9RC5Lr0J7VHUssBSv5sKDhxnla_GSs` 即为文件夹 ID，根目录为 `root` 。

## 阿里云盘

### 根文件夹 ID

文件夹的唯一标识，可从浏览器地址栏获取。

以备份盘为例，目标文件夹对应URL `https://www.alipan.com/drive/file/all/backup/623fbf465be853af1a0f44eb8842bdf76d477da9`
最后面的一串字符 `623fbf465be853af1a0f44eb8842bdf76d477da9` 即为文件夹 ID，根目录为 `root` 。

## 115网盘

### 根文件夹 ID

文件夹的唯一标识，可从浏览器地址栏获取。

目标文件夹对应 URL `https://115.com/?cid=3105489445714151206&offset=0&tab=&mode=wangpan` 查询参数 `cid` 的值
`3105489445714151206` 即为文件夹 ID，根目录为 `0` 。

## 123云盘

### 根文件夹 ID

文件夹的唯一标识，可从浏览器地址栏获取。

目标文件夹对应 URL `https://www.123pan.com/?homeFilePath=15374274,15374275` 查询参数 `homeFilePath` 的值中以逗号分隔的最后一串数字
`15374275` 即为文件夹 ID，根目录为 `0` 。

## WebDAV

### 服务器地址

如果你希望添加某个特定的文件夹而不是根目录，请在服务器地址后面拼接对应的路径。比如上游提供的 WebDAV 服务器地址为
`https://dav.jianguoyun.com/dav/` ，目标文件夹为 `/Voices/DLsite` ，对应的服务器地址请填写
`https://dav.jianguoyun.com/dav/Voices/DLsite` 。

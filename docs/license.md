# 许可证

Neokikoeru 是付费的专有软件，为了支持软件的持续开发和维护，你需要购买并激活许可证使用。

## 如何购买？

你可以通过[爱发电](https://ifdian.net/a/vscodev)平台来购买 [Neokikoeru 许可证](https://ifdian.net/item/91e62d426f5b11ecbd5c52540025c377)，支付成功后许可证密钥会自动通过私信发送给你。

> [!TIP]
> 你可以免费阅览/播放不超过 30 MB 的文件以评估 Neokikoeru 的各项功能指标，请先试用再购买。

## 如何激活？

前往**仪表盘**->**系统信息**->**激活许可证**，输入你收到的许可证密钥然后点击**确定**即可。

![activate-license](/images/activate-license.webp)

> [!IMPORTANT]
> 激活软件需要联网，设备时间和网络时间的误差不得超过 90 秒。每个许可证密钥最多可激活 3 台设备，支持[反激活](https://www.voicehub.top/license)。

# 常见问题

## 更新后报错，程序无法启动怎么办？

Neokikoeru 偶尔会出现破坏性更新，如果更新后程序运行异常，你可以运行 `neokikoeru prune` 命令清除应用数据重新初始化。

> [!TIP]
> 对于 Windows 用户，如果你不会命令行，请在资源管理器地址栏输入 `%AppData%` 后回车，找到 `neokikoeru` 文件夹删除即可。

## 为什么媒体库没有数据？

Neokikoeru 是一个自托管的服务程序，软件本身不提供任何内容，媒体库资源需要你自行导入，详见[添加存储](/guide/storages)。

## 为什么刮削一直提示网络错误？

你所处的网络环境无法直连 DLsite，请设置 `HTTP_PROXY` 和 `HTTPS_PROXY` 这两个环境变量为你的代理地址。

## 为什么有些作品刮削不到，提示 `product not found`?

DLsite 的部分标签如`萝莉`等仅限日本 IP 访问，如果作品包含这些标签你需要将代理软件切换到日本节点。

## 能否给媒体库的作品添加字幕标识？

不能。为了兼容第三方云存储和提升性能，Neokikoeru 并不会递归获取作品文件夹下的所有内容，而是按需加载，程序在扫描入库时并不知道作品是否包含字幕。

## 能否录入非【音声・ASMR】类型的作品？

这是支持的，程序并没有对此进行限制，你可以录入 DLsite 贩售的任意作品。
